#🎵 Проєкт «Голос проти раку» | Андрій Герц
Благодійний музичний проєкт автора пісень, спрямований на підтримку людей, які борються з онкологічними захворюваннями, через реалізацію авторського контенту та просвітницьку діяльність.

🚀 Як це працює
Прослуховування та вибір: Відвідувачі можуть прослухати 82-секундні прев'ю треків у високій якості безпосередньо на сайті.

Пряма оплата: Процес покупки повністю автоматизований та інтегрований із системою Monobank E-commerce, що підтримує Google Pay, Apple Pay та оплату карткою.

Миттєве отримання: Після успішної транзакції покупець одразу отримує посилання на завантаження повної версії файлу з Google Drive.

Додатковий фонд: Проєкт монетизується за допомогою Google AdSense для збору додаткових коштів на підтримку онкохворих.

🛠 Технології
Frontend: HTML5, CSS3 (Modern UI), JavaScript (AOS, Google Analytics).

Backend: Python / Flask API, розгорнутий на платформі Render.

Сховище: Google Drive API для надійної доставки цифрового контенту.

Монетизація: Google AdSense (інтегровано через ads.txt та рекламні скрипти).
