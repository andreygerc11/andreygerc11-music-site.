# 🎵 Проєкт «Голос проти раку» | Андрій Герц

Це благодійний проєкт, створений для підтримки онкохворих через продаж авторської музики.

## 🚀 Як це працює
* **Сайт:** Користувач обирає трек на сайті.
* **Бот:** Після вибору треку користувач переходить у Telegram-бот для оплати.
* **Оплата та отримання:** Після підтвердження оплати, бот автоматично надсилає посилання на скачування з Google Drive.

## 🛠 Технології
* **Frontend:** GitHub Pages (HTML/JS)
* **Backend:** Python (Aiogram, Flask)
* **Hosting:** Render
* **Database:** Google Drive API
